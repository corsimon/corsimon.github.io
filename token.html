Informations Token

Token ERC-20

ERC-20 est une sorte de standard universel pour émettre un token. Cela permet de prévoir exactement comment le token se comportera
Cela apporte un certain avantage pour les wallets qui peuvent ajouter très aisément le support de nouveaux tokens, et pour les développeurs car cela allège considérablement leur travail

ICO

Une ICO (Initial Coin Offering) est une méthode de levée de fonds, fonctionnant via l'émission d'actifs numériques échangeables contre des cryptomonnaies durant la phase de démarrage d'un projet. Ces actifs numériques sont appelés tokens (voir au dessus)
Pour faire une ICO, il faut donc tout d'abord créer le token qui sera ensuite mis en vente aux potentiels investisseurs de manière décentralisé grâce à un smart-contract. L'idée du TD était de créer une ICO avec notre propre token sur le réseau de test Ethereum Ropsten.
Malheureusement, nous avons manqué de temps durant les TDs pour mettre à bien notre projet, vous pourrez tout de même trouver le début de smart-contract d'ICO que j'ai pu faire pour ce cours sur mon github (lien)
