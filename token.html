Token ERC-20

ERC-20 est une sorte de standard universel pour émettre un token. Cela permet de prévoir exactement comment le token se comportera
Cela apporte un certain avantage pour les wallets qui peuvent ajouter très aisément le support de nouveaux tokens, et pour les développeurs car cela allège considérablement leur travail
